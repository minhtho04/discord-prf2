# Jadoo Discord Corner

Landing page tĩnh cho GitHub Pages, với phong cách Discord neon và animation CSS/SVG. Không cần Node.js hay build tool.

## Tùy chỉnh nhanh

Trong `index.html`, thay các giá trị sau:

- `YOUR_NAME`: tên hiển thị.
- `YOUR_INVITE`: mã mời Discord, ví dụ `my-server` trong `https://discord.gg/my-server`.
- `YOUR_DISCORD`: Discord username hoặc ID để nút Copy hoạt động.

## Deploy với GitHub Pages

1. Tạo repository tên `YOUR_USERNAME.github.io` (ví dụ: `minji.github.io`).
2. Upload toàn bộ nội dung thư mục này vào root repository.
3. Vào **Settings → Pages**.
4. Chọn **Deploy from a branch**, branch `main`, folder `/(root)`, rồi bấm **Save**.
5. Website sẽ xuất hiện ở `https://YOUR_USERNAME.github.io` sau vài phút.

Giao diện tự đáp ứng điện thoại và tôn trọng cài đặt giảm chuyển động của người xem.
